# Contribuindo

1. **Crie uma branch**: `git checkout -b feat/nome-da-feature`
2. **Rode o lint/testes** localmente: `pre-commit run -a` e `pytest`
3. **Abra PR** com descrição clara. Relacione issues (ex.: `Closes #12`).

## Convenções
- Python 3.11+
- Black (formatação) + Ruff (lint)
- Testes com Pytest
